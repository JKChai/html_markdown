  2. Write-Log

    .SYNOPSIS
    Log messaging function
    .DESCRIPTION
    Write log message to the specified file with date & log type. Both messages & log type are required for this function to work, where log type only accept INFO, ERROR, & DEBUG information. In addition, both $LogFullpath & $ScriptName identifies are required to be declared & has to be defined in the Global scope. $LogFullpath is the variable name that consists of absolute path of the target log file while $ScriptName is the variable name that provides the script name when the script is invoked at runtime.
    .PARAMETER message
    Type: [String]
    Information provided to log
    .PARAMETER level
    Field: [DEBUG|INFO|ERROR]
    Type: [String]
    Adopted from Python logging standards https://docs.python.org/3/howto/logging.html
    INFO is used when the log message is log as part for informing purposes
    ERROR is used when the code is deem to be broken and is normally used in try-catch block
    DEBUG is used when the code is on debugging purposes to identify issues or for testing
